# Tutor Agent Tools

| Tool | What It Does |
|---|---|
| Web search | Pull up-to-date references and official docs with sources. |
| Course file search (RAG) | Search syllabus, slides, assignments, rubrics, and notes to stay course-aligned. |
| Code execution (Python) | Run snippets to verify answers, test edge cases, and generate plots. |
| Math solver (CAS) | Do symbolic algebra/calculus steps (or via SymPy through code execution). |
| Learner progress DB | Store and retrieve goals, misconceptions, mastery, and past attempts. |
| Quiz/exercise generator + grader | Create practice at a target difficulty and grade with rubric-style feedback. |
| Citation/snippet extractor | Attach short cited snippets from web/notes to reduce hallucinations. |
| Study-plan generator | Build a weekly plan from deadlines, topics, and available time. |
| Academic integrity guard | Detect “do my homework” and switch to hints/scaffolding mode. |
| Diagramming/visualization | Generate graphs, ERDs, automata, circuits, and concept maps. |
| LaTeX formatter | Produce clean math/solution formatting and normalize student math input. |
| Worksheet/notebook exporter | Export a session to a worksheet or `.ipynb` (optionally withholding solutions). |
| Units/consistency checker | Track units and catch dimensional mistakes in STEM problems. |
